boot-starter如何工作？
利用maven的pom依赖描述机制,starter依赖的jar都是空的,没有任何东西,实际起作用的是和jar一起下载的pom.xml文件
和jar在同一个文件夹内。在IDEA中只能看到空的jar确看不到和jar一起下载的pom.xml文件。要到本地仓库中才能看到。
ctrl+左键鼠标点击依赖也能看到pom.xml的内容

例如 spring-boot-starter
下载的jar:spring-boot-starter-log4j2-2.6.6.jar
和jar一起下载的pom.xml:spring-boot-starter-log4j2-2.6.6.pom


springBoot与webServer的方式
1.spring boot 直接管理webServer
2.webServer 管理 springBoot, SpringServletContainerInitializer
